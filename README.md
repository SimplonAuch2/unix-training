### Upgrade

Don't forget to keep Ubuntu/Debian up to date :wink: :

```shell
sudo apt-get update;
sudo apt-get upgrade;
sudo apt-get dist-upgrade;
sudo apt-get autoremove;
sudo apt-get autoclean;
```
